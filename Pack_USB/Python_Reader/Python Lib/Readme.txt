Pour pouvoir lancer le script Python "Decodage.py" il faut suivre les étapes suivantes.
Les fichiers donnés sont compatibles Windows 7 32 bits. Pour une autre config, aller sur :
matplotlib.org/users/installing.html
- Installer Python (avec le .msi)
- Installer Matplotlib, Numpy et Pyparsing (avec les .exe)
- Copier/Coller dans votre dossier d'installation Python > Lib > le fichier six.py (librairie six) et le dossier dateutil (autre librairie).
